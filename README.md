# Smart Construction Site Manager

## Project Overview
Smart Construction Site Manager is a desktop application project planned in C# for construction site management.

The goal of this system is to help manage workers, tasks, attendance, materials, and project progress in one organized platform.

## Problem Statement
Construction sites often face management problems such as:
- unclear task assignments
- poor attendance tracking
- unorganized material records
- difficulty monitoring project progress

This project aims to solve these problems with a structured management system.

## Target Users
- Admin
- Site Manager
- Worker

## Planned Features
- User login
- Role-based access
- Worker management
- Project management
- Task assignment
- Attendance tracking
- Material tracking
- Progress monitoring

## Technologies
- C#
- .NET
- Desktop GUI
- Database
- GitHub

## Project Status
Currently in planning and design phase.
